# ES6-MathOperation

> Just a very simple example using ECMAScript 6

If you have never used class in javascript, here's an example :)

```
const math = new MathOperation();
```

###### addition
```
math.addition(20, 10);
```

###### subtraction
```
math.subtraction(30, 10);
```

###### multiplication
```
math.multiplication(40, 20);
```

###### division
```
math.division(50, 20);
```
