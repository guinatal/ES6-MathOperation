# ES6-MathOperation

> Just a very simple example using ECMAScript 6

You who have never used class in javascript. Here's a simple example :)

```
const math = new MathOperation();
```

```
math.addition(20, 10);
```

```
math.subtraction(30, 10);
```

```
math.multiplication(40, 20);
```

```
math.division(50, 20);
```
